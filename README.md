# SuperTrunfoEmC-Modulo01

Este repositório contém um projeto simples desenvolvido durante a faculdade com o objetivo de praticar conceitos básicos da linguagem C.
O foco principal é o aprendizado de lógica de programação e estruturas fundamentais da linguagem.

Super Trunfo em C

Projeto simples inspirado no jogo Super Trunfo, utilizando estruturas condicionais e de repetição para comparar atributos entre cartas.

Conceitos praticados

if / else

switch

while

Operadores relacionais e lógicos

Entrada e saída de dados

Objetivo: praticar tomada de decisão no código e controle de fluxo.
